🏨 THE PENTOUZ Hotel Booking Widget Test Page

This page demonstrates the embeddable booking widgets for THE PENTOUZ Hotel Management System.

1. Inline Widget

This widget embeds directly into the page content, perfect for dedicated booking pages.

<div id="booking-widget-main-booking-widget"></div> <script> (function() { var script = document.createElement('script'); script.src = 'http://localhost:4000/widget.js'; script.async = true; script.onload = function() { HotelBookingWidget.init({ widgetId: 'main-booking-widget', type: 'inline', theme: { primaryColor: '#3b82f6', secondaryColor: '#1d4ed8', textColor: '#1f2937', borderRadius: '8px', fontFamily: 'Inter' }, openInNewTab: false, bookingUrl: 'http://localhost:3000/booking' }); }; document.head.appendChild(script); })(); </script> <!-- End Hotel Booking Widget -->

2. Popup Widget

This widget appears as an overlay popup when triggered, great for call-to-action buttons.

<button onclick="HotelBookingWidget.show()">Book Now</button> <script> (function() { var script = document.createElement('script'); script.src = 'http://localhost:4000/widget.js'; script.async = true; script.onload = function() { HotelBookingWidget.init({ widgetId: 'popup-promo-widget', type: 'popup', theme: { primaryColor: '#dc2626', secondaryColor: '#fef2f2', textColor: '#1f2937' } }); }; document.head.appendChild(script); })(); </script>

3. Floating Widget

This widget appears as a floating button in the bottom-right corner, perfect for persistent booking access.

<script> (function() { var script = document.createElement('script'); script.src = 'http://localhost:4000/widget.js'; script.async = true; script.onload = function() { HotelBookingWidget.init({ widgetId: 'mobile-floating-widget', type: 'floating', autoShow: true, autoShowDelay: 3000, theme: { primaryColor: '#059669', secondaryColor: '#ecfdf5' } }); }; document.head.appendChild(script); })(); </script>

4. Widget Analytics & Tracking

All widgets automatically track impressions, clicks, conversions, and user behavior for detailed analytics.

// Widgets automatically track: - Impressions (when widget loads) - Clicks (when user interacts) - Conversions (when booking is completed) - Device/Browser information - Geographic data - Session behavior // Manual tracking (optional): HotelBookingWidget.track('custom_event', { customData: 'value', userId: 'guest123' });
