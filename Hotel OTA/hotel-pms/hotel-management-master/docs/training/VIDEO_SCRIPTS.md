[Screen: Show inherited setting indicator]

**Narrator**: "Individual settings show blue 'i' icons when they're inherited from the group."

**Narrator**: "Benefits of inheritance:**
✓ Automatic synchronization - no manual updates needed
✓ Guaranteed consistency across properties
✓ Reduces configuration errors
✓ Saves hours of manual work"

[Screen: Flexibility note]

**Narrator**: "And here's the best part: properties can still override specific settings when they need to be different. It's standardization with flexibility."

### General Production Guidelines

**Video Format**:
- Resolution: 1080p (1920x1080) minimum
- Aspect Ratio: 16:9
- Frame Rate: 30fps
- Audio: Stereo, 48kHz
- Duration: As specified per video
- Format: MP4 (H.264 codec)

- Screen recording: 1920x1080 capture
- Microphone: Professional quality, noise-canceled
- Lighting: Consistent, no flicker
- Background: Quiet, no distractions

**Visual Style**:
- Brand colors: THE PENTOUZ blue (#1E40AF), white, gray
- Fonts: Clean, readable (16pt minimum for text)
- Cursor: Highlighted for visibility
- Callouts: Arrows, circles, boxes for emphasis
- Transitions: Smooth, professional (fade, dissolve)

**Screen Captures**:
- Always use consistent test data across all videos
- Properties named and located consistently
- Pre-configure test environment before recording
- Clear cache/cookies for clean UI
- Hide personal information
- Use demo credentials, not real

**Audio Guidelines**:
- Clear, professional narration
- Conversational tone but authoritative
- Pace: Not too fast, allow time for viewers to process
- Pauses: After important points
- Background music: Subtle, non-distracting (optional)

**On-Screen Text**:
- Use sparingly, for emphasis only
- Large, readable font
- High contrast
- Appear/disappear smoothly
- Stay on screen long enough to read (minimum 3 seconds)

**Accessibility**:
- Closed captions: Accurate, synchronized
- Audio descriptions: For key visual elements
- Keyboard navigation demos: Slow and clear
- Color: Don't rely solely on color for meaning
